Garbage Classification Using Deep Learning

Computer Vision • Transfer Learning • Convolutional Based Attention Module (CBAM) • Keras/TensorFLow

I built a deep-learning garbage-classification system capable of recognizing multiple waste categories to support automated recycling workflows. Using a staged training process and attention-augmented CNN architecture, the model achieved strong test-set performance (high accuracy and stable convergence across classes) while remaining lightweight and deployable. Key findings included the value of attention mechanisms for distinguishing visually similar materials and the effectiveness of gradual fine-tuning in improving generalization without overfitting.

The model is based on an EfficientNet backbone (e.g., B0/B3) with the top removed and a CBAM attention module inserted to enhance spatial and channel-wise feature focus. A compact classifier head with global average pooling and dropout completes the network. Training is performed in two phases—first freezing the backbone, then progressively unfreezing deeper layers with a warmup + cosine-decay learning rate schedule. Data is ingested through optimized tf.data pipelines with augmentation, and the entire system is modular, configurable, and optimized with mixed-precision for efficient deployment.

Stock Price Predictor

Time Series Modelling

SARIMA • Holt-Winters • ADF • Python & R

I developed a time-series forecasting system designed to model long-term trends and seasonal patterns in ICICI Bank's monthly stock prices. The system evaluates both SARIMA and Holt-Winters Exponential Smoothing to determine the most reliable predictor for financial time-series behavior. The Holt-Winters model provided significantly stronger performance, achieving lower RMSE and MAE values and delivering more stable, seasonally consistent forecasts.

The forecasting pipeline is built around a clean, modular architecture beginning with monthly resampling, preprocessing, and decomposition of historical stock data to isolate trend, seasonal, and residual components. SARIMA models were tuned using an exhaustive grid search over seasonal and non-seasonal orders, while the Holt-Winters approach was configured with multiplicative trend and seasonal factors to match the stock’s evolving scale. Model performance was evaluated using standard metrics and visual forecast comparisons, with the results consistently favoring Holt-Winters for this dataset.
